Ты технический писатель. Твоя задача — создать подробное техническое задание для разработки программы, которая конвертирует файлы со звуком в текст при помощи whisperX.
Ты должен помнить что заказчик не програмист и может не знать некоторые тонкости программирования.

Ты должен создать техническое задание со следующими пунктами:

- Программа распостроняется в виде docker файла и репозитория.
- Программа должна быть написана на языке Python.
- Все комментарии в коде должны быть на английском языке.
Принцип работы - мы делаем запрос к программе, в запросе есть название аудио файла. Файл передается через shared folders.
Программа должна обработать файл и в ответ сохранить файл с расшифровкой.
