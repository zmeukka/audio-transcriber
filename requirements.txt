# Инструкция по ручной установке модели Whisper/Faster-Whisper для Windows
# 1. Перейдите на страницу модели, например: https://huggingface.co/Systran/faster-whisper-small
# 2. Скачайте все файлы модели вручную (weights, config, vocabulary и др.).
# 3. Поместите скачанные файлы в папку: models/faster-whisper-small
# 4. Для запуска whisperx используйте параметр --model_dir:
#    python -m whisperx input/sample.oga --output_dir output --output_format txt --language ru --device cpu --model_dir models/faster-whisper-small
# 5. Если возникает ошибка с симлинками, выполните в PowerShell:
#    $env:HF_HUB_DISABLE_SYMLINKS=1
# 6. Все зависимости должны быть установлены через requirements.txt:
#    pip install -r requirements.txt
#
# После этих шагов whisperx будет использовать локальные файлы модели и не обращаться к HuggingFace Hub.
#
# Для других моделей используйте соответствующую папку и ссылку.
#
# Пример для запуска:
# python -m whisperx input/sample.oga --output_dir output --output_format txt --language ru --device cpu --model_dir models/faster-whisper-small
#
# Пример для запуска с поддержкой float32 (решает ошибку ValueError: Requested float16 compute type...)
# python -m whisperx input/sample.oga --output_dir output --output_format txt --language ru --device cpu --model_dir models/faster-whisper-small --compute_type float32
#
# Если требуется диаризация или VAD, убедитесь, что соответствующие модели также скачаны вручную.
